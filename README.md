# codeBulk

This repository contains a collection of code solutions and examples, primarily focused on data structures and algorithms implemented in Java.

## Overview

The repository is organized into directories, each representing a specific topic or problem-solving area.

## Contents

*   **AdjacentRemove**: Contains code related to removing adjacent duplicates from a string.
*   **Backtracking**: Includes examples and solutions using backtracking algorithms.
*   **BinaryTree**: Focuses on binary tree data structure implementations and algorithms.
*   **DS_Preperation**: Contains Data Structures preparation related code
*   **Encoding**: Includes different types of Encoding algorithm
*   **HackerRanks**: Solutions for various HackerRank problems.
*   **LeetCode #mayChallenge**: Solutions for LeetCode May Challenge problems.
*   **LinkListDuplication**: Code for Link List Duplication
*   **LinkedListProblem**: Contains code for solving various linked list problems.
*   **MaxDepth**: Includes examples for finding the maximum depth of a tree.
*   **Recursion**: Examples demonstrating the use of recursion.
*   **RoomRequired**: Code for Room Required problem
*   **StringProblem**: Contains solutions to various string-related problems.
*   **TeamBuilding**: Includes examples for Team Building problem
*   **TreeProblem**: Contains solutions for tree-related problems.
*   **hackerlandRadio**: Code related to hackerland Radio problem
*   **leetCode30**: Includes examples for leetCode30 problem
*   **lexiographicNumSort**: Code for lexiographic Number Sorting problem
*   **miniOperation**: Includes examples for mini Operation problem
